# Test calculator

Requirements:
**Qt::Core**
**Qt::Quick**

Проект розбит на підпапки:
**library** - сама бібліотека обчислення виразу за вхідним рядком.
**app** - gui частина для тестування бібліотеки у віконному інтерфейсі.

Також до репозиторія на всяк випадок додав папку з бінарниками на Windows x64 (win64_build). Виконуваний файл - testcalculator.exe.
Віконний інтерфейс робив через QML.
Відповідно зовнішні залежності проекта це Qt::Core та Qt::Quick.
